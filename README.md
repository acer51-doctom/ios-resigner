# ios-resigner
A Rust app to resign all of your sideloaded apps incase you need to do it manually.
This is a vibecoded app since I didn't want to put too much time into it. I will still maintain it but it's not gonna have new crazy features. **This is based off of idevice_tools's and iloader's work, rewritten in Rust.**

## Why?
Why? Because instead of using the extremely slow iLoader and having trouble with the UDID issues, I decided to make this and share it to others, in case it'll ever become useful to society one day.
That's it, lol.
